# 🤖 Simple AI Search Chatbot (Python)

## 📌 Project Overview
This project is a **simple AI chatbot** built using Python that searches through a local text file (`data.txt`) and returns relevant answers.

It uses a **basic keyword matching technique** to simulate how AI search systems work.  
Endee is included in the environment but not used in this simplified version.

---

## 🚀 Features
- 🔍 Keyword-based search
- 📂 Reads data from a local file
- 💬 Interactive command-line chatbot
- ⚡ Fast and lightweight
- 🧠 Beginner-friendly AI project

---

## 🛠️ Tech Stack
- Python 🐍
- File Handling
- String Matching

---

## ⚙️ How It Works
1. Loads data from `data.txt`
2. Takes user input as a query
3. Searches matching lines using keyword comparison
4. Returns top results
5. Displays output in chat format
